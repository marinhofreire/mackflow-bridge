# mackflow-bridge

API bridge Node.js + TypeScript (Express) com conectores Cabme e Z-API Z-Pro.

## Requisitos
- Node.js 18+

## Configuração
1. Copie o arquivo .env.example para .env e preencha os valores.
2. Instale dependências: npm install

## Executar em desenvolvimento
npm run dev

## Build e produção
npm run build
npm start

## Endpoints
- GET /health
- GET /admin/smoke
